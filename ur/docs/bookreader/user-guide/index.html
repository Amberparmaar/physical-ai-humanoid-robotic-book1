Getting Started​

Accessing the Application​

  1. Open your web browser (Chrome, Firefox, Safari, Edge)
  2. Navigate to the BookReader application URL
  3. The application will load with the library view by default

Initial View​

When you first open the application, you'll see:

  • Header with application name
  • Library view showing available books
  • Footer with copyright information

Reading Experience​

Opening a Book​

  1. From the library view, browse the book cards
  2. Click on any book card to open it
  3. The book will load in the reading area with clean typography
  4. You can now start reading the content

Navigating Content​

  • Scroll through the book content using your mouse wheel or touchpad
  • On touch devices, swipe up/down to scroll
  • Reading progress is automatically saved as you read

Selecting Text​

  • Click and drag to select text in the reading area
  • Selected text will be highlighted
  • A floating translation button will appear near the selection

Translation Features​

Translating Text​

  1. Select any paragraph or text in the book
  2. A floating "Translate to Urdu" button will appear
  3. Click the button to get the Urdu translation
  4. The Translation Panel will open showing:
    • Original English text
    • Urdu translation (right-to-left text)
    • Controls to switch display modes

Translation Panel Features​

  • Side-by-Side View: See both languages next to each other
  • Toggle View: Switch between English and Urdu
  • Close Button: Close the translation panel
  • Mode Switcher: Toggle between side-by-side and toggle view

Language Switching​

  • Use the Language Toggle button to cycle through language preferences:
    • English Only
    • Urdu Only
    • Both Languages
  • The reading area will update immediately to reflect your selection

Customization Options​

Adjusting Font Size​

  1. Click on the Settings icon (usually in the header or reading area)
  2. Use the slider to adjust font size between 12px and 36px
  3. The text size will update immediately in the reading area
  4. Your preference is saved automatically

Theme Selection​

You can switch between light and dark themes:

  1. Open the Settings panel
  2. Click the theme toggle button
  3. The application will switch to your selected theme
  4. Themes are saved automatically and will persist between sessions

Saving Preferences​

All your preferences (font size, theme, language setting) are automatically saved in your browser's local storage. These settings will be applied the next time you visit the application.

Browsing Books​

Library View​

  • Books are displayed as cards in a responsive grid
  • Each card shows:
    • Book title
    • Author name
    • Translation availability indicator
    • Character count
  • Hover over cards to see more details

Book Information​

  • Title and Author: Clearly displayed on each card
  • Translation Indicator: Shows if Urdu translation is available
  • Character Count: Provides an idea of the book length

Searching and Filtering​

(If implemented) Use the search bar at the top of the library to find specific books by title or author.

Troubleshooting​

Common Issues​

Translation Not Working​

  • Ensure you have selected text before clicking the translation button
  • Check your internet connection
  • Try refreshing the page if translations are not loading
  • Make sure JavaScript is enabled in your browser

Display Issues​

  • Zoom level might be affecting readability: use Ctrl+0 (or Cmd+0) to reset
  • Try clearing your browser cache: Settings → Privacy → Clear browsing data
  • Ensure your browser is up to date
  • Disable browser extensions that might interfere with the application

Reading Progress Not Saving​

  • Make sure JavaScript is enabled
  • Check if your browser blocks localStorage (settings → privacy → cookies)
  • Try using a different browser if the issue persists

Browser Compatibility​

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+
  • For best experience, use the latest version of your preferred browser

FAQ​

Q: How do I access Urdu translations? A: Select any text in the reading area and click the floating "Translate to Urdu" button that appears.

Q: Can I see both English and Urdu side-by-side? A: Yes! The Translation Panel includes a mode switcher to toggle between side-by-side and toggle view.

Q: How are my preferences saved? A: Your preferences are stored in your browser's local storage and will persist between sessions.

Q: What book formats are supported? A: The application currently supports plain text, PDF, and EPUB formats.

Q: Can I use this on my mobile device? A: Yes! The application is fully responsive and works on mobile devices, tablets, and desktops.

Q: Is my reading progress saved? A: Yes, your reading progress is automatically saved as you read through a book.

Q: How do I switch between languages? A: Use the Language Toggle button to cycle through English only, Urdu only, or both languages.

Q: Can I adjust the text size? A: Yes, use the settings panel to adjust font size from 12px to 36px.

Q: What if I see an error message? A: Try refreshing the page first. If issues persist, check your internet connection or try using a different browser.

Q: Are there keyboard shortcuts? A: Currently, the application focuses on mouse/touch interactions for optimal reading experience, but standard browser shortcuts (like Ctrl+F for find) work normally.

Support​

If you encounter any issues not covered in this guide:

  1. Refresh your browser page
  2. Clear your browser's cache and cookies
  3. Try using an incognito/private browsing window
  4. Check your internet connection
  5. Contact support through the provided channels
